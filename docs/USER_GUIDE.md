## はじめに

オヤデキは、親御さん（60代以上）のデジタル困りごとを解決するLINE Botです。

### できること
- 📷 **救急箱**: スマホ画面のスクショを送ると、何が起きているか説明＋操作手順を案内
- 💬 **下書き提案**: お子さんへの返信に困ったら、メッセージをコピペで3案提案
- 📑 **契約台帳**: サブスクや公共料金の契約情報を家族で共有

### 友だち追加
LINE公式アカウント「オヤデキ」を友だち追加してください。

## 基本機能

### 🆘 救急箱（スクショ解析）

**使い方**
1. スマホで困っている画面のスクリーンショットを撮る
2. オヤデキにその画像を送る
3. AIが画面を解析して、状況説明＋操作手順を返信

**対応シーン**
- パスワード忘れ画面 → 再発行手順を案内
- SMSコード未着 → 原因切り分け
- アプリ更新ダイアログ → ストア遷移案内
- 詐欺警告ポップアップ → 「無視してOK！」と安心させる

**返信の見方**
```
⚠️ 警告: （詐欺や危険がある場合のみ）
📱 状況: 画面から読み取れる状況
1️⃣ 手順1
2️⃣ 手順2
3️⃣ 手順3
📞 通話誘導（複雑な場合）
```

---

### 💬 下書き提案

**使い方**
1. お子さんから来たLINEメッセージをコピー
2. オヤデキにそのまま貼り付けて送信
3. 返信案A/B/Cから選んで「コピー」をタップ
4. お子さんとのトークに貼り付けて送信

**返信例**
```
A: 忙しい中ありがとう！体調には気をつけてね。
B: そうなんだ〜！また落ち着いたら話聞かせてね。
C: 無理しないでね。何かあったらいつでも電話して！
💬 質問: 最近どう？
📞 通話: 今度ゆっくり電話しようか！
```

## 契約台帳機能

### 📑 台帳に登録する

**方法1: 救急箱から登録**
1. 契約書や請求書の画面をスクショして送信
2. 救急箱の返信下部「📑 これを台帳に登録」をタップ
3. AIが抽出した情報を確認
4. 「この内容で登録」をタップ

**方法2: 直接画像を送信**
1. 契約書・請求書の画像を送信
2. 「📑 これを台帳に登録」をタップ
3. 内容を確認して登録

**登録される情報**
- サービス名（例: Netflix、東京電力）
- 種類（公共料金/サブスク/保険/通信/その他）
- 月額（推定）
- ID等（メールアドレスや契約番号）
- メモ（解約方法など）

> ⚠️ **パスワードは登録されません**（安全のため）

---

### 📋 台帳を見る

**コマンド**: `台帳` と送信

**表示内容**
- 📊 サマリー（登録件数、月額合計）
- 各契約のカード（スワイプで閲覧）

**未確認バッジ**
- 7日以上確認していない契約には「⚠️ 未確認」バッジが表示
- 「確認する」ボタンで確認済みに更新

---

### 🔗 グループに共有する

**使い方**
1. `台帳` と送信
2. サマリーカードの「🔗 グループに共有」をタップ
3. 共有リンクが生成される（30日間有効）
4. リンクをグループに貼り付け

**グループに表示される内容**
- 登録件数
- 月額合計
- 主要サービス一覧（上位5件）
- 「詳細を見る」ボタン

**詳細ページ**
- 共有リンクをタップすると専用ページで全契約を確認可能
- パスワードは表示されない

> 💡 グループ内で「台帳」と送信すると、サマリーのみ表示されます（詳細は個人チャットで確認）

---

### 📥 エクスポートする

**使い方**
1. `台帳` と送信
2. サマリーカードの「📥 エクスポート」をタップ
3. CSV形式のデータが返信される
4. コピーしてメモ帳やExcelに貼り付け

**出力フォーマット**
```csv
サービス名,種類,月額,ID等,メモ,最終確認日
"Netflix","subscription","1490","example@email.com","毎月15日引落","2026/1/20"
```

> ⚠️ パスワードは含まれません

---

### 📊 週報（自動配信）

**条件**: 台帳に5件以上登録している場合

**配信タイミング**: 毎週日曜 19:00

**内容**
- 登録件数
- 月額合計
- 未確認件数（7日以上確認していない契約）

## 設定

### ⚙️ 設定画面を開く

Rich Menuの「設定」をタップ、または以下のURLにアクセス:
https://oyadeki-liff.deno.dev/settings

### 設定項目

| 項目 | 説明 | 選択肢 |
|-----|------|-------|
| メタファーテーマ | 例え話のスタイル | ツェーゲン金沢、大相撲、なし |
| メタファーON/OFF | 例え話を使うかどうか | ON / OFF |
| トーン | 返信の口調 | 丁寧語 / カジュアル / 温かい |
| NG語 | 使ってほしくない言葉 | カンマ区切りで入力 |

## トラブルシューティング

### 「ユーザー登録が見つかりません」と表示される

**原因**: 初回設定が完了していない

**対処**:
1. 設定画面を開く: https://oyadeki-liff.deno.dev/settings
2. LINEでログイン
3. 設定を保存

---

### 画像を送っても反応しない

**原因**: 画像の読み込みに時間がかかっている

**対処**:
- 10秒ほど待つ
- それでも反応しない場合は、もう一度送信

---

### グループで反応しない

**仕様**: グループでは以下のキーワードがある場合のみ反応します
- 「オヤデキ」「おやでき」
- 「使い方」「ヘルプ」「help」
- 「台帳」

画像は常に反応します。

---

### 共有リンクが開けない

**原因**: 有効期限切れ（30日）

**対処**: 新しい共有リンクを発行してください
