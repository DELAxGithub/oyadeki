# オヤデキ（Oyadeki）
## 親世代のためのAIアシスタント

---

# オヤデキとは？

オヤデキは、スマホで困ったときに助けてくれる **LINEのお友だち** です。

**覚えることは1つだけ：**
> 困ったら写真を送る

これだけで、AIがやさしく解決方法を教えてくれます。

---

# こんなお悩みありませんか？

## スマホのトラブル
- 変な警告画面が出て怖い
- 操作方法がわからない
- 詐欺かどうか判断できない

## 日常の管理
- 毎月何に払っているか把握できない
- 見た映画やテレビ番組を忘れてしまう
- いらないものを売りたいけど出品が面倒

## 家族との連携
- 子どもに何度も同じことを聞くのが申し訳ない
- でも困ったときは頼りたい

**→ オヤデキがすべて解決します！**

---

# できること一覧

## 1. スマホの「困った」を解決

### 使い方
1. 困っている画面のスクリーンショットを撮る
2. オヤデキに写真を送る
3. AIが状況を説明し、やることを教えてくれる

### 特徴
- **詐欺警告を検知** → 赤い警告で危険を知らせる
- **やることは3ステップ以内** → シンプルでわかりやすい
- **例え話で説明** → 好きなスポーツに例えて説明（設定可能）

### 例：詐欺画面を送った場合

```
⚠️ 警告：これは詐欺の可能性があります！

📱 状況：
「ウイルスに感染しました」という警告画面です。
これは偽の警告で、詐欺サイトです。

【やること】
1️⃣ ホームボタンを押す
2️⃣ ブラウザを閉じる
3️⃣ もう開かない

📞 難しければ、お子さんに連絡しましょう
```

## 2. 契約台帳

### 毎月の支払いを一目で把握

- 携帯代、電気代、Netflix...
- 何にいくら払っているか一覧で見える
- 「未確認」マークで確認漏れを防止

### 登録方法
1. 契約書や請求書の写真を送る
2. AIが内容を読み取る
3. 「台帳に登録」ボタンを押すだけ

### 便利機能
- **週1回のサマリー配信** → 支払い状況をお知らせ
- **家族と共有** → 子どもがリモートで確認できる
- **CSVエクスポート** → Excelで管理したい人向け

---

## 3. 見たものログ

### 映画・テレビ・本を自動記録

1. テレビ画面や本の表紙を撮影
2. オヤデキに送る
3. 自動で作品名を判定
4. 星で評価をつける（1〜5）

### 対応メディア
- 🎬 映画
- 📺 テレビ番組
- ⚽ スポーツ中継
- 📚 本・雑誌
- 🎵 音楽

### あとから振り返り
- 「あの番組なんだったっけ？」
- 「最近見た映画は？」
- → メニューの「見たもの」で確認！

---

## 4. 出品サポート（メルカリ用）

### いらないものを売りたいとき

1. 「売る」と送る（または商品の写真をいきなり送る）
2. AIの質問に答える（購入時期、傷の有無など）
3. メルカリ用のタイトルと説明文が完成！

### 生成される内容
- 商品タイトル（40文字以内）
- 商品説明文
- おすすめカテゴリ
- 状態（傷なし/やや傷ありなど）

### 完成例

```
【タイトル】
任天堂 Switch 有機ELモデル ホワイト 美品

【説明文】
任天堂 Switch 有機ELモデルです。
2022年に購入、週末のみ使用。
動作確認済み。
付属品：本体、ドック、Joy-Con左右、ACアダプター
目立った傷や汚れはありません。
```

長押しでコピー → メルカリに貼り付け！

## 5. タスク管理（NEW!）

### 毎朝「今日のやること」が届く

子どもが設定したタスクが、毎朝LINEでプッシュ通知されます。

### 配信イメージ

```
☀️ 今日のやること

□ 電気の検針票を探す
   └ 見つけたら写真送ってね

□ ガスの請求書を確認
   └ 同じ棚にあるかも

□ 水道の契約番号をメモ
   └ 検針票の右上に記載

[完了を報告] [全部見る]
```

### 特徴
- **1日3件だけ** → 無理のないペースで進められる
- **完了報告がカンタン** → ボタンを押すだけ
- **進捗を子どもと共有** → 離れていても見守れる

### 活用例
- 実家の片付け・断捨離
- ライフラインの契約確認
- 定期的な健康チェック

# メニュー画面

画面下に4つのボタンがあります。

```
┌─────────────┬─────────────┐
│  🚑 救急箱   │ 📑 契約台帳  │
├─────────────┼─────────────┤
│  📺 見たもの │  📦 売る    │
└─────────────┴─────────────┘
```

| ボタン | できること |
|--------|-----------|
| 🚑 救急箱 | 使い方・ヘルプを表示 |
| 📑 契約台帳 | 登録した契約の一覧を見る |
| 📺 見たもの | 記録した映画・番組を振り返る |
| 📦 売る | メルカリ出品モードを開始 |

※ タスクは「タスク」と送信するか、毎朝届くプッシュ通知から確認できます

# 子どもとつながる安心

## グループLINEで見守り

- 親子のグループにオヤデキを招待
- 親が写真を送ると、子どもにも見える
- 詐欺警告などの緊急時にすぐ対応

## 「電話で聞く」ボタン

- 困ったときはワンタップで電話
- 子ども側に通知が届く
- スムーズにバトンタッチ

## 台帳の共有機能

- 契約情報を家族にシェア
- パスワードは絶対に共有されない
- 離れていても支払い状況を把握

# 設定のカスタマイズ

## 話し方を選べる

| 設定 | 例 |
|------|-----|
| 丁寧 | 「ホームボタンを押してください」 |
| カジュアル | 「ホームボタン押してね」 |
| 温かい | 「大丈夫ですよ、ホームボタンを押しましょうね」 |

## 例え話のテーマ

好きなスポーツや趣味に例えて説明

- サッカー（ツェーゲン金沢など）
- 野球
- 大相撲
- 将棋
- なし（例え話を使わない）

## NG語の設定

使ってほしくない言葉を設定できます

- 「頑張って」→ プレッシャーに感じる方
- 「簡単」→ できないと落ち込む方

# よくある質問

## Q. 写真を送っても返事が来ない
**A.** 10秒ほど待ってください。それでも来なければ再送信。

## Q. スクリーンショットの撮り方は？
**A.**
- iPhone（ホームボタンなし）: 電源 + 音量上を同時押し
- iPhone（ホームボタンあり）: 電源 + ホームボタンを同時押し

## Q. パスワードは保存される？
**A.** いいえ。安全のため、パスワードは絶対に保存しません。

## Q. 料金はかかる？
**A.** 無料です。
