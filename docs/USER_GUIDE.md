# 第1部：親御さん向けガイド

---

## 1. オヤデキとは

オヤデキは、スマホで困ったことがあったときに助けてくれるLINEのお友だちです。

### できること

| やりたいこと | やり方 |
|-------------|--------|
| スマホで困った | 画面の写真を送る |
| テレビで見た番組を記録したい | テレビの写真を送る |
| いらないものを売りたい | 「売る」と送ってから商品の写真を送る |
| 契約を確認したい | 画面下の「契約台帳」をタップ |

### 覚えることは1つだけ

**「困ったら写真を送る」**

これだけ覚えてください。あとはオヤデキがやってくれます。

---

## 2. 写真を送るだけ

### 困ったとき

1. 困っている画面のスクリーンショットを撮る
   - iPhone: 電源ボタン + 音量上ボタン を同時押し
   - iPad: 電源ボタン + ホームボタン を同時押し

2. オヤデキにその写真を送る

3. オヤデキが「何が起きているか」と「やること」を教えてくれる

**例：こんな返信が来ます**

```
⚠️ 警告：これは詐欺の可能性があります！

📱 状況：
「ウイルスに感染しました」という警告画面が表示されています。
これは偽の警告で、詐欺サイトです。

【やること】
1️⃣ ホームボタンを押してください
2️⃣ Safariアプリを完全に閉じてください
3️⃣ このサイトには二度とアクセスしないでください

📞 難しければ、お子さんに電話してみてください
```

**返信の下にボタンが出ます**
- 「わかった！」→ 解決したときに押す
- 「電話で聞く」→ 子どもに電話したいときに押す
- 「📑 台帳に登録」→ 契約画面だった場合に押す

---

### テレビや映画を見たとき

1. テレビ画面や映画のポスターの写真を撮る
2. オヤデキに送る
3. 何を見たか自動で判定して、記録してくれる

**例：こんな返信が来ます**

```
🎬 映画

ゴジラ-1.0
監督: 山崎貴
2023年

⭐ 評価をつけてください
[⭐1] [⭐2] [⭐3]
[⭐4] [⭐5] [スキップ]
```

星をタップすると記録されます。あとで「見たもの」から振り返れます。

---

### いらないものを売りたいとき（メルカリ用）

1. いきなり商品の写真を送る（または「売る」と送ってから）
2. オヤデキ「これを出品しますか？」→「はい」
3. 「どこで買いましたか？」「傷はありますか？」と聞かれるので答える
4. メルカリ用のタイトルと説明文が自動で作られる

**例：こんな返信が来ます**

```
📦 出品文が完成！

【タイトル】
任天堂 Switch 本体 有機ELモデル ホワイト 美品

【説明文】
任天堂 Switch 有機ELモデルです。
2022年に購入し、週末のみ使用していました。
動作確認済み。
付属品：本体、ドック、Joy-Con左右、ACアダプター
目立った傷や汚れはありません。

カテゴリ: ゲーム機
状態: 目立った傷なし

↑ 長押しでコピーできます！
```

完了後もしばらくの間「いくらで売れる？」などと質問すれば答えてくれます！
文章を長押しでコピーして、メルカリアプリに貼り付けてください。

## 3. リッチメニューの使い方

画面の下に4つのボタンがあります。

```
┌──────────────┬──────────────┐
│   📑 契約台帳  │  📖 見たもの  │
├──────────────┼──────────────┤
│   💬 相談履歴  │  📲 シゴデキ  │
└──────────────┴──────────────┘
```

### 📑 契約台帳

登録した契約（Netflix、電気、保険など）の一覧を見られます。

**見られる情報**
- 登録しているサービスの一覧
- 月々いくら払っているかの合計
- 最後に確認した日

**未確認マーク**
7日以上確認していない契約には「⚠️ 未確認」と表示されます。
「確認する」を押すと消えます。

---

### 📖 見たもの

写真を送って記録した視聴履歴を見られます。

**見られる情報**
- 見た映画・番組・スポーツの一覧
- つけた評価（星の数）
- いつ見たか

「あの番組なんだったっけ？」というときに便利です。

**感想戦（アフタートーク）**
「見た」記録をした直後（30分以内）なら、「主演の俳優は？」「ロケ地は？」など、その作品についての質問をそのまま送ればBotが答えてくれます！

---

### 💬 相談履歴

過去に困って相談したことの一覧を見られます。

**見られる情報**
- どんな画面で困ったか
- そのとき教えてもらった手順
- 解決したかどうか

「前も同じことがあったな」というときに振り返れます。

---

### 📲 シゴデキ

家族共有のリマインダーアプリ「シゴデキ」を開きます。

## 4. よくある質問（親御さん向け）

### Q. 写真を送っても返事が来ない

**A.** 10秒ほど待ってみてください。画像の解析に時間がかかることがあります。
それでも来ない場合は、もう一度送ってみてください。

---

### Q. どうやってスクリーンショットを撮るの？

**A.**
- **iPhone（ホームボタンなし）**: 右側面のボタン + 左側面の音量上ボタン を同時に押す
- **iPhone（ホームボタンあり）**: 右側面のボタン + ホームボタン を同時に押す
- **iPad**: 上部のボタン + ホームボタン を同時に押す

撮った写真は「写真」アプリに保存されています。

---

### Q. パスワードは登録されないの？

**A.** 安全のため、パスワードはオヤデキに保存されません。
パスワードは今まで通り、iCloudのパスワード管理をお使いください。

---

### Q. グループLINEでも使える？

**A.** 使えます。グループでは、写真を送ると自動で反応します。
文字だけのメッセージには、「オヤデキ」「台帳」などのキーワードがないと反応しません。

---

### Q. 設定を変えたいときは？

**A.** 「設定」とメッセージを送ってください。設定画面が開きます。
（普段は触らなくて大丈夫です。お子さんに頼んでください。）

## 5. 初期設定の手順

### 5.1 友だち追加

1. 親御さんのスマホで、LINE公式アカウント「オヤデキ」を友だち追加
2. QRコード or 検索で追加

### 5.2 初期設定（子どもが親のスマホで操作）

友だち追加直後に、以下のメッセージが届きます：

```
🤖 オヤデキへようこそ！
   まず設定をお願いします
   [設定を開く]
```

1. 「設定を開く」をタップ
2. LINEでログイン（親のアカウントでログイン）
3. 各項目を設定（詳細は次章）
4. 「個人情報の取り扱いについて同意します」にチェック
5. 「保存する」をタップ

### 5.3 親子グループへの追加（推奨）

1. 親子のグループLINEに「オヤデキ」を招待
2. グループでも写真を送れば反応する
3. 子ども側も親の相談内容を把握できる

### 5.4 リッチメニューの確認

トーク画面下部にリッチメニュー（4ボタン）が表示されているか確認。
表示されていない場合は、トーク画面下部の「≡」をタップして展開。

## 6. 設定項目の詳細

### 設定画面の開き方

- **初回**: 友だち追加時に自動表示
- **あとから**: 「設定」と送信 → リンクをタップ

### 6.1 例え話のテーマ

**概要**: AIの返信に使う例え話のテーマを設定

**設定例**:
- ツェーゲン金沢（サッカー）
- 大相撲
- 将棋
- 野球
- なし（例え話を使わない）

**使用例（ツェーゲン金沢の場合）**:
```
⚠️ レッドカード！これは詐欺のファウルプレーです！

📱 状況：
相手チーム（詐欺師）がPKを狙っています。

【ディフェンス】
1️⃣ ゴールを守る（ホームボタンを押す）
2️⃣ ボールをクリア（ブラウザを閉じる）
3️⃣ 次の試合に備える（もう開かない）
```

**推奨**: 親御さんが好きなスポーツや趣味があれば設定。なければ「なし」。

### 6.2 例え話を使う（ON/OFF）

**ON**: 上記のような例え話を使った返信
**OFF**: 通常の説明文で返信

**推奨**: 親御さんの好みに合わせる。迷ったらOFF。

### 6.3 話し方（トーン）

| 設定 | 説明 | 例 |
|-----|------|-----|
| 丁寧 | です・ます調 | 「ホームボタンを押してください」 |
| カジュアル | くだけた表現 | 「ホームボタン押してね」 |
| 温かい | 優しい表現 | 「大丈夫ですよ、ホームボタンを押しましょうね」 |

**推奨**: 「丁寧」または「温かい」

### 6.4 使わないでほしい言葉（NG語）

**概要**: AIに使ってほしくない言葉を設定

**入力方法**: カンマ区切りで入力
```
頑張って, 大丈夫, 簡単
```

**よくあるNG語**:
- 「頑張って」→ プレッシャーに感じる方
- 「簡単」→ できないと落ち込む方
- 「お年寄り」→ 呼ばれたくない方

### 6.5 個人情報の取り扱いについて同意

**必須項目**。チェックしないと保存できません。

**同意内容**:
- 送信した画像はAI（Gemini）で解析されます
- 契約情報はデータベースに保存されます
- パスワードは保存されません

## 7. 機能リファレンス

### 7.1 救急箱（困りごと対応）

**トリガー**: 画像送信 → AI判定「困りごと」

**処理フロー**:
1. 画像をGemini Visionで解析
2. 状況説明を生成
3. 解決手順（最大3ステップ）を生成
4. 危険度判定（詐欺・ウイルス警告など）
5. Flex Messageで返信

**危険度判定**:
- 詐欺系: 赤背景の警告バナーを表示
- 通常: 警告なし

**返信構造**:
```
[⚠️ 警告バナー]（危険時のみ）
[📱 状況説明]
[【やること】1️⃣ 2️⃣ 3️⃣]
[📞 通話誘導]（複雑時のみ）
[ボタン: わかった！ / 電話で聞く / 台帳に登録]
```

**Postback処理**:
- `vision=xxx&result=understood` → 解決ログ記録
- `vision=xxx&result=call` → 通話誘導ログ記録
- `action=propose_ledger&msgId=xxx` → 台帳登録フローへ

---

### 7.2 メディアログ（視聴記録）

**トリガー**: 画像送信 → AI判定「メディア」

**対応メディアタイプ**:
| タイプ | アイコン | 判定例 |
|-------|---------|--------|
| 映画 | 🎬 | 映画のポスター、Netflix画面 |
| テレビ番組 | 📺 | 番組表、録画リスト |
| スポーツ | ⚽ | 試合中継、スポーツニュース |
| 音楽 | 🎵 | CDジャケット、Spotify画面 |
| 本 | 📚 | 本の表紙、Kindle画面 |

**処理フロー**:
1. 画像からメディア情報を抽出（タイトル、出演者、年など）
2. 評価入力を促すFlex Messageを返信
3. 星評価（1-5）またはスキップを受付
4. media_logsテーブルに記録

**一覧表示**:
- トリガー: リッチメニュー「見たもの」または「見た」と送信
- 最新20件をカルーセル形式で表示

---

### 7.3 契約台帳

**登録方法**:

**A. 救急箱から登録**
1. 契約画面のスクショを送信
2. 救急箱返信の「📑 台帳に登録」をタップ
3. AI抽出情報を確認
4. 「この内容で登録」をタップ

**B. 直接登録**
1. 契約書・請求書の画像を送信
2. AIが契約情報と判定した場合、登録提案

**保存される情報**:
| フィールド | 説明 | 例 |
|-----------|------|-----|
| service_name | サービス名 | Netflix |
| category | 種類 | subscription/utility/insurance/telecom/other |
| monthly_cost | 月額（円） | 1490 |
| account_identifier | ID・契約番号 | example@gmail.com |
| note | メモ | スタンダードプラン、毎月15日引落 |
| last_confirmed_at | 最終確認日 | 2026-01-27 |

**保存されない情報**:
- パスワード（セキュリティ上、絶対に保存しない）

**一覧表示**:
- トリガー: リッチメニュー「契約台帳」または「台帳」と送信
- サマリー（件数・合計金額）+ 各契約カードのカルーセル

**未確認バッジ**:
- 条件: last_confirmed_at が7日以上前
- 表示: オレンジ背景の「⚠️ 未確認」バッジ
- 解除: 「確認する」ボタンをタップ

**グループ共有**:
1. 「🔗 グループに共有」をタップ
2. 共有トークン（16文字）を生成
3. 有効期限: 30日間
4. 共有ページURL: `/share/{token}`
5. パスワードは表示されない

**CSVエクスポート**:
- 「📥 エクスポート」をタップ
- フォーマット: `サービス名,種類,月額,ID等,メモ,最終確認日`

**週報**:
- 条件: 5件以上登録
- 配信: 毎週日曜 19:00
- 内容: 件数、合計金額、未確認件数

### 7.4 相談履歴

**トリガー**: リッチメニュー「相談履歴」

**表示内容**:
- 過去の救急箱相談の一覧
- 各相談: タイトル（自動生成）、日付、やったこと、ステータス

**用途**:
- 同じ問題が再発したときの参考
- 子どもが親の困りごとを把握

**データソース**: usage_logs テーブル（action_type = vision_help）

---

### 7.5 出品サポート（メルカリ用）

**トリガー**:
- 「売る」「出品」「メルカリ」と送信
- または、商品の写真をいきなり送信（「出品しますか？」と聞かれる）

**処理フロー**:
1. 出品モード開始
2. ユーザーが商品写真を送信
3. Gemini Visionで商品情報を抽出
4. 不足情報があればBotから質問（対話モード）
5. 必要な情報が揃ったら、タイトル・説明文・カテゴリ・状態を生成
6. 完了後、感想戦モードへ移行（30分間）

**生成内容**:
| 項目 | 説明 |
|-----|------|
| タイトル | メルカリ用の40文字以内のタイトル |
| 説明文 | 商品の状態、購入時期、付属品など |
| カテゴリ | メルカリのカテゴリ |
| 状態 | 新品/未使用に近い/目立った傷なし/やや傷あり/傷あり |

**制限**:
- 出品モードは5分間でタイムアウト
- 1回の写真送信で1商品

---

### 7.6 ヘルプ・使い方

**トリガー**: 「使い方」「ヘルプ」「help」と送信

**返信内容**:
- 基本的な使い方（写真を送るだけ）
- コマンド一覧（台帳、見た、売る、設定）
- スクリーンショットの撮り方

## 8. トラブルシューティング

### 8.1 画像を送っても反応しない

**原因1**: 処理に時間がかかっている
- 対処: 10秒待つ

**原因2**: タイムアウト（3秒以上）
- 症状: 「少々お待ちください...」と表示される
- 対処: 後から詳細な返信が来る場合あり。来なければ再送信

**原因3**: サーバーエラー
- 症状: 「⚠️ すみません、うまく考えられませんでした」
- 対処: 時間をおいて再送信

### 8.2 グループで反応しない

**仕様**: グループではテキストメッセージに以下のキーワードが必要
- 「オヤデキ」「おやでき」
- 「使い方」「ヘルプ」「help」
- 「台帳」「契約台帳」「ledger」
- 「見た」「見たもの」「メディアログ」

**画像は常に反応します**（キーワード不要）

### 8.3 設定が保存できない

**原因1**: 同意チェックがない
- 対処: 「個人情報の取り扱いについて同意します」にチェック

**原因2**: LINEログインしていない
- 対処: LIFFでログインしてから再試行

### 8.4 「ユーザー登録が見つかりません」

**原因**: 初回設定が未完了

**対処**:
1. 「設定」と送信
2. 設定画面を開く
3. 各項目を入力して保存

### 8.5 共有リンクが開けない

**原因**: 有効期限切れ（30日）

**対処**: 新しい共有リンクを発行

### 8.6 台帳に登録されない

**原因1**: 画像が契約情報と判定されなかった
- 対処: 契約書・請求書の全体が写るように撮り直す

**原因2**: 登録確認ボタンを押していない
- 対処: 「この内容で登録」をタップ

### 8.7 メディアが認識されない

**原因**: 画面が暗い、ぼやけている、小さすぎる

**対処**:
- 画面を明るくして撮影
- テレビに近づいて撮影
- 番組表やポスターを撮影

## 9. 運用のコツ

### 9.1 親御さんへの教え方

**教えることは1つだけ**:
「困ったら写真を送って」

細かい機能は覚えなくてOK。使っているうちに自然と覚えます。

**最初に一緒にやること**:
1. 試しにテレビの写真を送ってもらう
2. 評価ボタンを押してもらう
3. 「見たもの」で記録を確認

### 9.2 グループの活用

**推奨構成**:
- 親 + 子（+ 他の家族）のグループに招待
- 親が写真を送ると、子どもも内容を把握できる
- 「わかった！」「電話で聞く」の選択も見える

**メリット**:
- 親が困っていることをリアルタイムで把握
- 詐欺警告などの緊急時にすぐ対応できる

### 9.3 契約台帳の定期確認

**推奨運用**:
1. 最初に主要な契約を5〜10件登録
2. 週報が届くようになる
3. 「未確認」が溜まったら一緒に確認

**確認のタイミング**:
- 週末に「台帳」で一覧を見る
- 「確認する」を押して最終確認日を更新

### 9.4 設定の調整

**見直しタイミング**:
- 返信がわかりにくいと言われたとき
- 例え話が合わないと感じたとき
- NGワードを追加したいとき

**変更方法**:
1. 親のスマホで「設定」と送信
2. 設定画面を開く
3. 子どもが代わりに設定を変更
