# オヤデキ LINE インターフェース デザイン発注要件書

## 概要

**プロダクト名**: オヤデキ
**コンセプト**: 親御さんのデジタル困りごとを子どもと一緒に解決するLINEボット
**ターゲット**: 60代以上の親御さん（主要）、30〜40代の子ども世代（補助）

---

## 1. リッチメニュー

### 仕様
- **サイズ**: 2500px × 810px（LINE公式仕様）
- **ボタン数**: 3つ（横並び均等配置）
- **タップ領域**: 各ボタン 48px以上確保

### ボタン構成

| ボタン | ラベル | 機能 |
|--------|--------|------|
| 左 | 見た！ | 視聴したメディアの記録を表示 |
| 中央 | 設定 | パーソナライズ設定画面へ |
| 右 | 通話したい | 子どもへの電話を促す |

### デザイン方針
- **視認性重視**: 大きめアイコン + 短いラベル
- **コントラスト**: 4.5:1以上（アクセシビリティ）
- **トーン**: 温かみのある親しみやすさ（堅すぎない）

### カラーバリエーション（3種類）
1. **レッド系** - 活発・元気なイメージ
2. **ブラック系** - シンプル・モダン
3. **ゴールド系** - 上品・信頼感

## 2. Flex Message テンプレート

### 共通デザインルール

#### カラーパレット
```
プライマリ（成功・確認）: #1DB446（LINE Green）
セカンダリ（補足）      : #666666
警告（注意喚起）        : #E65100（オレンジ）
エラー（危険）          : #CC0000（赤）
背景ソフト              : #F0FFF4（薄緑）, #FFF8E1（薄黄）, #FFE0E0（薄赤）
```

#### タイポグラフィ
- **見出し**: Bold, 16-18px
- **本文**: Regular, 14px, 行間1.5
- **補足**: 12px, グレー（#888888）

#### ボタン
- **高さ**: 48px以上（タップしやすさ）
- **角丸**: 8px
- **スタイル**: Primary（塗り）/ Secondary（枠線）/ Link（テキストのみ）

### 2.1 救急箱回答メッセージ

**用途**: スマホの画面写真を送ると、解決手順を返す

```
┌─────────────────────────────────┐
│ ⚠️ 警告バナー（危険時のみ）      │  背景: #FFE0E0
│ 「これは詐欺の可能性が...」      │  文字: #CC0000
├─────────────────────────────────┤
│ 📱 状況説明                      │
│ 「〇〇の画面が表示されています」 │
├─────────────────────────────────┤
│ 【やること】                     │
│ 1️⃣ ホームボタンを押す           │
│ 2️⃣ ブラウザを閉じる             │
│ 3️⃣ 設定を確認                   │
├─────────────────────────────────┤
│ 📞 難しければお子さんに電話を    │  ※複雑時のみ表示
├─────────────────────────────────┤
│ [わかった！] [電話で聞く]        │  ボタン2つ
│ [📑 これを台帳に登録]            │  ※契約画面時のみ
└─────────────────────────────────┘
```

**ポイント**:
- 手順は最大3つまで（スクロール防止）
- 各手順は絵文字で視覚的に番号付け
- 警告バナーは目立つ赤系で危機感を演出

### トーン
- 親しみやすい丸みのあるデザイン
- 線の太さ: 2-3px（視認性確保）
- カラー: 単色 or 2色まで

## 6. 納品物リスト

### 必須
1. **リッチメニュー画像** × 3種類（2500×810px, PNG）
2. **Flex Message モックアップ** × 7種類（Figma or XD）
3. **LIFF設定画面 UI** × 1画面
4. **共有ページ UI** × 1画面
5. **アイコンセット** × 7種類（SVG）

### オプション
- ローディングアニメーション
- エラー状態のUI
- 空状態のUI（データがない時）

---

## 7. 技術的な制約

### LINE Flex Message
- 最大バブル数: 12個（カルーセル）
- 最大ボタン数: 4個/バブル
- 画像は外部URL必須（Base64不可）
- フォント指定不可（システムフォント使用）

### アクセシビリティ
- コントラスト比: 4.5:1以上
- タップ領域: 48×48px以上
- フォントサイズ: 最小14px
