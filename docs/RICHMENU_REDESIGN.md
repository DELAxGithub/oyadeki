# リッチメニュー リデザイン要件

## 背景
タスク管理機能を追加したため、リッチメニューの構成を変更する必要があります。

---

## 現状（2x2 / 4ボタン）

```
┌─────────────┬─────────────┐
│  🚑 救急箱   │ 📑 契約台帳  │
│ 困ったときは │ 契約書・ハガキ │
├─────────────┼─────────────┤
│  📺 見たもの │  📦 売る    │
│ テレビ・映画 │ メルカリ出品 │
└─────────────┴─────────────┘
```

**サイズ**: 1200 x 810 px

---

## 変更後（案A: 2x3 / 6ボタン）

```
┌─────────────┬─────────────┐
│  🚑 救急箱   │ ✅ タスク    │
│ 困ったときは │ 今日のやること │
├─────────────┼─────────────┤
│  📑 契約台帳 │ 📺 見たもの  │
│ 契約書・ハガキ│ テレビ・映画 │
├─────────────┼─────────────┤
│  📦 売る    │ ⚙️ 設定     │
│ メルカリ出品 │ カスタマイズ │
└─────────────┴─────────────┘
```

**サイズ**: 1200 x 1215 px（高さ1.5倍）

### ボタン配置の意図
| 位置 | 機能 | 理由 |
|-----|------|------|
| 左上 | 救急箱 | 最重要機能。目につきやすい位置 |
| 右上 | タスク | 毎日使う新機能。救急箱と同列 |
| 中段左 | 契約台帳 | 定期確認用 |
| 中段右 | 見たもの | エンタメ記録用 |
| 左下 | 売る | 頻度低め |
| 右下 | 設定 | たまにしか使わない |

---

## 変更後（案B: 上大下小 / 5ボタン）

```
┌───────────────────────────┐
│       🚑 救急箱            │
│     困ったときはココ        │
├─────────┬────────┬────────┤
│ ✅ タスク│ 📑 契約 │ 📺 見た │
├─────────┼────────┼────────┤
│ 📦 売る │ ⚙️ 設定 │（空き） │
└─────────┴────────┴────────┘
```

### メリット
- 救急箱を最も目立たせられる
- 緊急時にすぐタップできる

### デメリット
- ボタンサイズが不均一でタップしにくい可能性

---

## 変更後（案C: 3x2 / 6ボタン・横長）

```
┌──────────┬──────────┬──────────┐
│ 🚑 救急箱 │ ✅ タスク │ 📑 契約  │
├──────────┼──────────┼──────────┤
│ 📺 見た  │ 📦 売る  │ ⚙️ 設定  │
└──────────┴──────────┴──────────┘
```

**サイズ**: 1200 x 540 px（高さ2/3）

### メリット
- コンパクトでトーク画面を邪魔しない
- 均等サイズでタップしやすい

### デメリット
- ボタンが小さくなり、文字が見づらい可能性

---

## 推奨案

**案A（2x3 / 6ボタン）を推奨**

理由：
1. 現状と同じ正方形に近いボタンでタップしやすい
2. 各機能に十分なスペースを確保できる
3. 高齢者でも押しやすいサイズ
4. 将来の機能追加にも対応しやすい

---

## 各ボタンの詳細仕様

| ボタン | アイコン | ラベル | サブラベル | アクション |
|--------|---------|--------|-----------|-----------|
| 救急箱 | 🚑 | 救急箱 | 困ったときはココ | テキスト送信「使い方」 |
| タスク | ✅ | タスク | 今日のやること | LIFF起動 `/tasks/{userId}` |
| 契約台帳 | 📑 | 契約台帳 | 契約書・ハガキ | LIFF起動 `/ledgers/{userId}` |
| 見たもの | 📺 | 見たもの | テレビ・映画 | LIFF起動 `/media/{userId}` |
| 売る | 📦 | 売る | メルカリ出品 | テキスト送信「売る」 |
| 設定 | ⚙️ | 設定 | カスタマイズ | LIFF起動 `/settings` |

---

## デザイン要件

### カラー
- 背景: #FFFFFF（白）
- ボーダー: #D4AF37（ゴールド）2px
- ラベル文字: #06C755（LINEグリーン）
- サブラベル文字: #888888（グレー）

### フォント
- ラベル: Noto Sans JP Bold, 60px
- サブラベル: Noto Sans JP Regular, 24px

### アイコン
- サイズ: 100px
- スタイル: Apple絵文字風（既存踏襲）or カスタムアイコン

### ファイル形式
- PNG（透過なし）
- 解像度: 1200 x 1215 px（案Aの場合）
